== Description ==

anchorage is a throwback to the happy days of yore, with a single-column, content-first layout.  The masthead and all your nasty widgets are tucked away in a show/hide menu.  Additional widgets, the blog tagline, and another menu are displayed in the footer.

== Installation ==

Manual installation:

1. Upload the `anchorage` folder to the `/wp-content/themes/` directory

Installation using "Add New Theme"

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for `anchorage`
3. Click the `Install` button to open the theme's repository listing
4. Click the `Install` button

Activation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
